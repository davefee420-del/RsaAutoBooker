# RSA Auto Booker v3.0 Flawless 🇮🇪

**The ultimate Irish driving test slot finder with glassmorphism UI.**

## Features
- **3 Scan Modes**: Stealth (25-40s), Normal (12-20s), Aggressive (5-8s)
- **Auto-Click**: Clicks first available slot automatically
- **Auto-Book**: Completes full booking flow
- **Instant Notifications**: Push notification the moment a slot appears
- **Rate-Limit Protection**: Auto-detects blocks, switches to stealth
- **Glassmorphism UI**: Premium transparent Irish-themed design
- **Sound + Vibration Alerts**: Never miss a slot

## Setup
1. Upload all files to GitHub repo
2. Add your `gradle-wrapper.jar` to `gradle/wrapper/`
3. Push → Actions builds APK automatically

## Usage
1. Log in to MyRoadSafety
2. Go to your booking page
3. Tap yellow play button → starts scanning
4. When slot found: **Notification → Auto-click → Auto-book**
